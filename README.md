📘 Repositório de Lógica de Programação em Portugol (Visualg)

Este repositório contém minhas atividades, exercícios e práticas de lógica de programação desenvolvidas utilizando Portugol no Visualg, como parte dos estudos do curso de Algoritmos e Lógica de Programação.

🎯 Objetivo

O objetivo deste repositório é:

Praticar os conceitos fundamentais de lógica de programação

Desenvolver raciocínio lógico e resolução de problemas

Criar uma base sólida para aprender linguagens de programação mais avançadas

🧠 Conteúdos Abordados

Durante os exercícios, foram trabalhados os seguintes conceitos:

📥 Entrada e saída de dados

🔢 Variáveis e tipos de dados

➕ Operadores aritméticos e lógicos

🔀 Estruturas condicionais (se, senão)

🔁 Estruturas de repetição (para, enquanto, repita)

📊 Vetores (arrays)

📦 Procedimentos e funções

🛠️ Ferramentas Utilizadas

Visualg — interpretador de Portugol utilizado para executar os algoritmos

Portugol — linguagem didática para ensino de lógica de programação
